# Lews Search - 本地化搜索起始页

一个美观、功能丰富的本地化搜索起始页，支持多种自定义选项和功能。

## 功能特性

- **搜索功能**：支持Google搜索和手气不错
  - 手气不错：直接跳转到第一个搜索结果
  - 快捷键：当搜索框有文字时按Shift+Enter快捷键触发手气不错
- **最近访问**：自动记录并显示最近访问的网页
- **主题切换**：支持白天模式、夜间模式和自动模式（跟随系统）
- **背景设置**：
  - 液体玻璃效果（磨砂背景）
  - 本地图片上传作为背景
- **响应式设计**：适配不同屏幕尺寸
- **语音搜索**：支持语音输入搜索
- **新手教程**：首次使用时显示，手把手教用户使用网页
- **提示功能**：
  - 电脑版：悬停时显示操作提示
  - 手机版：点击时显示操作提示

## 如何使用

1. 克隆或下载本项目到本地
2. 打开 `index.html` 文件在浏览器中
3. 将其设置为浏览器的默认起始页：
   - Chrome：设置 > 外观 > 起始页 > 打开特定页面或一组页面 > 添加 > 选择 `index.html` 文件路径
   - Firefox：设置 > 主页 > 自定义 URLs > 输入 `index.html` 文件路径
   - Edge：设置 > 启动 > 打开特定页面 > 添加 > 选择 `index.html` 文件路径

## 自定义设置

点击右上角的设置图标（齿轮图标）打开设置面板：

1. **背景设置**：选择液体玻璃效果或本地图片作为背景
2. **主题设置**：选择白天模式、夜间模式或自动模式

## 技术栈

- CSS3

## 浏览器兼容性

支持所有现代浏览器，包括：
- Chrome
- Firefox
- Edge
- Safari

## 本地存储

所有设置和最近访问记录都存储在浏览器的本地存储中，确保刷新页面后设置不会丢失。

## 注意事项

- 语音搜索功能仅在支持 Web Speech API 的浏览器中可用
- 本地图片上传会将图片转换为 base64 格式存储在本地存储中，请注意存储空间限制
- 最近访问记录最多显示10条，固定的项目不计入限制
